this collection of projects is showing a simplified and limited role of logistics in a retail market.

A running demonsration will 3 warehouses with populated Inventories. Each Warehouse will have 3 stores within Same Day Transfer range with it's own Inventory. If a Store needs more Inventory, then a Bill of Lading will be created to transfer the Inventory from one or more Warehouses to one or more Stores. Once all Stops have been made, the Bill of Lading will be marked processed as the Line Entries related to the Bill of Lading will have already been marked processed.

There will also be a UI for warehouse and management employees to use for tracking Inventories and Bill of Ladings. This is authenticated by way of Auth0.

A second public UI may also be created for sales purposes.
