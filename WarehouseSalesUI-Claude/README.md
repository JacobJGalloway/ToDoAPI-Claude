WarehouseSalesUI-Claude

This is a simple web-based Sales UI for users to make purchases. The UI will show overall or regional/warehouse Inventory at current levels and expected incoming over X number of days. The closing of a sale will create an order with non-sensitive user info in an Order. and a BillOfLading for Logistics to fulfill. 

An Order is considered fulfilled when all of it's Inventory reductions have been completed through BillOfLadings related to the Order have completed processing.
